You can access SAM using MyWLCG and MyEGI web apps.
For a brief introduction to SAM please check the SAM Overview section.
Our current production release is SAM-Update-19.


The SAM team is composed by members from different organizations: AUTH, CERN, and SRCE. You can always contact the SAM team at sam-support@cern.ch.

All our code is released under the Apache 2 license. Most of the code is available at the CERN SAM repository.
