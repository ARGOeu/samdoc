SAM Doc : Deployment
This page last changed on Jul 02, 2012 by prodrigu.


This page describes the deployment strategy and the available configuration possibilities when deploying the SAM system. The SAM team distributes two node types: SAM-Nagios and SAM-Gridmon.

SAM-Nagios

A standard SAM-Nagios installations is composed by the following major components:

  • Nagios Core
  • Aggregator Topology Provider (ATP)
  • Profile Management Database (POEM)
  • Metric Result Store (MRS)
  • Nagios Config Generator (NCG)
  • Grid Probes

Installation

  • The recommended method for installing SAM-Nagios is using YUM. YUM must be configured to use the correct repositories as explained in the Installing SAM-Nagios guide.
  • The SAM-Nagios installation at CERN is done using quattor profiles.

Configuration

  • The SAM-Nagios node type can be configured as: NGI SAM-Nagios (for regions), Site SAM-Nagios (for one site), VO SAM-Nagios (for VOs)
  • The recommended method to configure a SAM-Nagios installation is using YAIM. YAIM configuration file must be edited as explained in the YAIM configuration section of the Installing SAM-Nagios guide.
  • The SAM-Nagios configuration installation at CERN is also done using YAIM. However YAIM configurations are retrieved from quattor profiles.

SAM-Gridmon

A standard SAM-Gridmon installations is composed by the following major components:

  • Nagios Core
  • Aggregator Topology Provider (ATP)
  • Profile Management Database (POEM)
  • Metric Result Store (MRS)
  • Nagios Config Generator (NCG)
  • Grid Probes
  • Availabilities Computation Engine (ACE)
  • MyWLCG/MyEGI Portal

Installation

  • The recommended method for installing SAM-Gridmon is using YUM. YUM must be configured to use the correct repositories as explained in the Installing SAM-Gridmon guide.
  • The SAM-Gridmon installation at CERN is done using quattor profiles.

Configuration

  • The recommended method to configure a SAM-Gridmon installation is using YAIM. YAIM configuration file must be edited as explained in the YAIM configuration section of the Installing SAM-Gridmon guide.
  • The SAM-Gridmon configuration installation at CERN is also done using YAIM. However YAIM configurations are retrieved from quattor profiles.
