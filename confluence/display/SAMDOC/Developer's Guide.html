This section provides information about SAM components to developers outside the SAM team.

  • Probes Development: SAM probes are developed by different groups and distributed by the SAM team. To allow an easy integration of probes a probes development guide is provided by the SAM team.
  • Web Services Specification: SAM offers a programmatic interface exposing a number of web services. These web services are available under MYWLCG and MYEGI.
  • Availability Computation: SAM computes availability and reliability numbers for services instances, services flavours, and site. This page expalins on how this component computes availability and reliability.
