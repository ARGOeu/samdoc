Introduction

The package provides set of probes which check that the number of incoming metrics to the MRS is in its average during the last hour .

Probes

The package provides the following set of probes:

  • MrsCheckDBInserts
  • MrsCheckDBInsertsDetailed
  • MrsCheckSpool

For more details, see Central Data Warehouse Monitoring.
