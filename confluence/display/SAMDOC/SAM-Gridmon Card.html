Introduction

This page presents the reference card for node sam-gridmon.

Services running

Service Apache
Service description  
Daemons httpd: /usr/sbin/httpd
Configuration /etc/httpd
Logs /var/log/httpd
Service supervisord
Service description Service managing execution of msg-consume2db services
Daemons httpd: /usr/bin/supervisord
Configuration /etc/supervisord.conf, /etc/supervisord.d
Logs /var/log/supervisor

Open ports

Port/Protocol Description
80/tcp Used by httpd (Required for the MyWLCG/MyEGI interface)
443/tcp Used by httpd (Required for the MyWLCG/MyEGI interface which requires authentication)

Cron jobs

Job Description
httpd-graceful Periodically sends a graceful event to httpd in order to update the ACLs
atp-sync Synchronization of ATP
ace-status Status computation in ACE
ace-availability Availability computation in ACE
dax-fts-summarizer Data Transfer Computation Engine - load FTS data to the Database
mrs-bootstrapper Re-populates the MRS configuration.
msg-consume2db-supervisor.chk Periodically checks if all msg-consume2db tasks are running and restart them if needed
poem-sync Synchronization of POEM
refresh-hosts Retrieve file nagios-roles.conf
refresh-roles Retrieve file hep-hosts.conf
voms-htpasswd Downloads the DN list from ATP for the configured VOs and creates an htpasswd file for apache. Runs periodically but only if the voms-htpasswd init script has run with "start" parameter.
