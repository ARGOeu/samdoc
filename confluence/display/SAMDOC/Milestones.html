SAM Doc : Milestones
This page last changed on Jul 16, 2012 by prodrigu.


This page provides an overview of the major milestones and target dates for their implementation. Milestones can contain multiple technical sub-tasks or improvements, which all need to be implemented in order for the milestone to be closed. Only improvements and tasks become part of the SAM releases (not the milestones), therefore there is no coupling between the SAM releases and the milestones. The milestones are sorted based on their priority.
